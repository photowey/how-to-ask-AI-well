# 致谢

这本书以现在的方式出现，有一个非常具体的触发点。

我想明确感谢 **Wu Ji**，以及他的论文 **《Trust Over Fear: How Motivation Framing in System Prompts Affects AI Agent Debugging Depth》**<sup><a href="/zh/appendices/sources#source-trust-over-fear-paper">[1]</a></sup>，以及相关 GitHub 项目 **NoPUA**<sup><a href="/zh/appendices/sources#source-nopua">[2]</a></sup>。

正是在阅读了这篇论文之后，我开始更认真地反思自己平时究竟是怎么向 AI 提问的。这篇论文不是这本书的全部来源，这本书也不等于这篇论文的扩写，但它的确提供了一个非常关键的触发：

**我们如何框定交给 AI 的工作，会影响它最终给回来的深度与质量。**

因此，这本书理应向该论文作者及这条研究线索明确致谢。

我也想感谢那些日常协作里的摩擦感。正是那些反复出现的浅层回答、过早收尾和“看起来已经完成”的虚假推进，逼着我把这些问题真正写清楚。没有这些不舒服的体验，这本书不会出现。
