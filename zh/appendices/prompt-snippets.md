# 附录 A：提问模板速查

这一节收集的是适合直接复制后再改写的短模板。

## 完整交付默认模板

```text
把这项任务按完整交付来处理。
先扫描完整的相关范围。
在当前范围内尽量一次性完成。
不要用“下一步建议”代替尚未完成的工作。
只有遇到真实阻塞时才允许停下。
```

## 深度排障模板

```text
这是一个深度优先的排障任务。
不要停在表面症状上。
请找到根因，检查直接相关的上下游组件，并明确写出验证方式。
```

## 先设计后实现模板

```text
先不要实现。
请先给出 2 到 3 套设计方案，说明权衡、变化轴和非目标，再推荐其中一套。
```

## 风险优先审查模板

```text
请围绕 bug、行为回归、边界情况、设计风险和缺失测试来审查这次改动。
把问题发现放在总结之前。
```
