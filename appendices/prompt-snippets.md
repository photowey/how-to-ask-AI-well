# Appendix A. Prompt Snippets

This appendix collects short patterns that can be copied and adapted quickly.

## Complete-delivery default

```text
Treat this as a complete-delivery task.
Scan the full relevant scope first.
Within the current scope, close the work in one pass.
Do not replace unfinished work with “next step” suggestions.
Only stop for real blockers.
```

## Deep debugging

```text
This is a depth-first debugging task.
Do not stop at the visible symptom.
Find the root cause, inspect directly related upstream and downstream components, and define verification.
```

## Design before implementation

```text
Do not implement yet.
First propose 2-3 design approaches, explain trade-offs, change axes, and non-goals, then recommend one.
```

## Risk-first review

```text
Review this change for bugs, regressions, edge cases, design risk, and missing tests.
Prioritize findings over summary.
```
